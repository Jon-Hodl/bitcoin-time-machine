# Bitcoin Time Machine

**Prove the 4-Year HODL Thesis Visually**

A client-side Bitcoin transaction analyzer that shows if your transaction has crossed the 210,000 block (4 year) threshold.

## Live Demo

🚀 **Try it now:** https://jon-hodl.github.io/bitcoin-time-machine-widget/

## Features

- ✅ Transaction lookup by TXID
- ✅ Block age calculation (blocks and time)
- ✅ Historical price lookup from [Bitcoin Price Database](https://jon-hodl.github.io/Bitcoin-Price/)
- ✅ Live current price from mempool.space
- ✅ Gain/loss calculation ($ and %)
- ✅ Anniversary price chart with volatility indicators
- ✅ Progress bar showing HODL progress
- ✅ Dull/vibrant mode switching at 4 years
- ✅ Mobile responsive
- ✅ Embeddable on any website

## How It Works

Bitcoin has never produced a loss after 4 years (210,000 blocks).

- **< 4 years**: Gray/dull interface — "Keep HODLing"
- **≥ 4 years**: Vibrant/golden interface — "The thesis holds true!"
